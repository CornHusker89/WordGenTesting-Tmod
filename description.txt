This mod adds a main-menu console and other features to enable automated world generation testing.
There is a multitude of built-in commands to execute manipulate tests, along with useful helper functions like creating worlds and taking screenshots.

NOTE - This is a mod intended to help modders! As a player, it is useless

Use the Github link found on this mod's steam page to find a guide on how to use this mod.
To report bugs, give feedback, or suggest new features join the discord, found on this mod's steam page.
